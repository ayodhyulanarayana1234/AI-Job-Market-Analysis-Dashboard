# AI Job Market Analysis Dashboard

## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing trends in the AI job market, including salary, job roles, skills demand, and hiring patterns.

## 🚀 Features
- Overview dashboard with KPIs and trends
- Deep analysis dashboard for salary, skills, and experience
- Interactive slicers for filtering data
- Advanced visuals like scatter plot, waterfall chart, and treemap

## Files Included
- Power BI Dashboard (.pbix)
- Dataset used for analysis
- Dashboard screenshots

## 💡 Insights
- AI and Machine Learning roles offer the highest salaries
- Python and ML are the most in-demand skills
- Experience significantly impacts salary growth

## 🛠 Tools Used
- Power BI
- DAX
- Data Cleaning
